Capacidad Concurrente Máxima

Límites máximos estimados basados en la infraestructura física contratada.

Límite de Ventas (RPS)

78 / seg

Con las 2 instancias de Render activas por auto-escalado y el pool de conexiones de Supavisor (50 conexiones), se procesan hasta 4,680 tickets por minuto en el momento del cierre.

Límite de Vendedores Activos

2,000+

Durante la operación normal diaria. Para cierres frenéticos simultáneos (últimos 5 minutos), el sistema soporta cómodamente hasta 400 vendedores enviando peticiones cada 5 segundos.

Garantías de Estabilidad

Mecanismos implementados para mantener la app en línea en caso de fallos.

Idempotencia

Locks atómicos vía scripts Lua en Redis que previenen que cobros o ventas duplicadas por doble-clic se registren dos veces.

Circuit Breaker

Si la cuota de Upstash se satura o se cae, el sistema bypassa Redis automáticamente durante 10 segundos para no bloquear la app.

Coalescing

Si 100 vendedores consultan resultados del sorteo al mismo tiempo, el servidor ejecuta una única consulta y comparte la respuesta.
